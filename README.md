# Stump engine SDL3

This is a game made for programming 4 to show game programming patterns.
Using beautiful c++

### patterns used

- Game Loop
- Update Method
- Command
- Observer/Event
- Components
- State
- Singleton
- Dirty Flag
- Service Locator

### Features

- Scene loading
- GameObjects
- Sounds
- Events / Event Listeners
- Resource caching
- InputMapping (Like unity)
- Textures
- Texture sprite sheets
- Components
